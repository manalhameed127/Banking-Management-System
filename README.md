# Banking Management System
Developed a Banking Management System in C++ using Object-Oriented Programming concepts such as inheritance, encapsulation, and polymorphism. Implemented account management, deposits, withdrawals, and transaction handling.

# OOP Banking Management System 

A C++ console-based Banking Management System developed as a 
university Object-Oriented Programming project.

## Features
- User account creation and management
- Company client account management
- Secure login system
- Account transactions and balance management
- Data persistence using JSON files
- Separate modules for User and Company clients

## OOP Concepts Used
- Classes and Objects (Account, UserClient, CompanyClient)
- Inheritance
- Encapsulation
- Polymorphism
- File handling with JSON

## Tech Stack
- Language: C++
- IDE: Visual Studio
- Data Storage: JSON files
